# Curriculum Year Plan

Pedagogically sequenced BJJ curriculum across three years.
Ordered by dependency, checkpoint stability, mechanic reuse, and duality.

Principle: maximize reusable mechanics early, low branching first,
safe failure topology, strong control foundations, progressive complexity.

NOT random positional coverage. NOT statistical optimization.
This is the optimal LEARNING traversal.

---

## Year 1: Beginner — Spine Roads

**Goal**: Learn all 7 fundamental positions, basic sweeps, one
submission from each dominant position, safe failure recovery.

**Principle**: Low branching. Clear cause/effect. Every road fails safely.
Every mechanic learned transfers to multiple roads later.

### Weeks 1–6: S1 — Closed Guard Hip Bump Sweep to Mount Armbar

**Fieldboard**: CGRD.CTRL → MNT.SWP → MNT.CTRL → MNT.SUB

**Prerequisites**: None (foundation road)

**Week breakdown**:
  - Week 1: CONTROL — Posture break in closed guard; Mount base stabilization
  - Week 2: TRANSITION — Hip bump mechanic: sit up, post hand, sweep over shoulder
  - Week 3: STABILIZATION — Secure mount before attacking — grapevines or heel hooks on hips
  - Week 4: SUBMISSION — Isolate arm → figure-four grip → hips high → lean back to extend
  - Weeks 5–6: DRILLING + positional sparring (this road only)

**Duality cycle**: Attack = sweep + armbar from guard. Defense = posture in guard, mount escape, armbar defense.

**Failure**: safe — Still in closed guard (safe)
**Mechanics learned**: hip escape, bridge, posture control, arm isolation

### Weeks 7–12: S5 — Open Guard Scissor Sweep to Mount

**Fieldboard**: OGRD.CTRL → MNT.SWP → MNT.CTRL

**Prerequisites**: None (foundation road)

**Week breakdown**:
  - Week 7: CONTROL — Collar grip + sleeve grip, shin across belly
  - Week 8: TRANSITION — Shin across belly → pull collar down → kick bottom leg → roll over
  - Week 9: STABILIZATION — Follow the sweep — post immediately on landing in mount
  - Weeks 10–12: DRILLING + positional sparring (this road only)

**Duality cycle**: Attack = scissor sweep. Defense = open guard passing, base maintenance.

**Failure**: safe — Still in open guard (safe — retain grips)
**Mechanics learned**: scissor motion, collar grip, sleeve control

### Weeks 13–18: S2 — Closed Guard Arm Drag to Back Control RNC

**Fieldboard**: CGRD.CTRL → BCTR.SWP → BCTR.CTRL → BCTR.SUB

**Prerequisites**: S1

**Week breakdown**:
  - Week 13: CONTROL — Posture break in closed guard; Seatbelt + hooks in back control
  - Week 14: TRANSITION — Arm drag: 2-on-1, pull arm across, hip out, climb behind
  - Week 15: STABILIZATION — Secure seatbelt grip before attacking choke
  - Week 16: SUBMISSION — Clear collar defense → slide choking arm under chin → lock figure-four → squeeze
  - Weeks 17–18: DRILLING + positional sparring (this road only)

**Duality cycle**: Attack = arm drag to back, RNC. Defense = posture in guard, back escape, RNC defense.

**Failure**: safe — Still in closed guard (safe)
**Mechanics learned**: arm drag, hip escape, seatbelt, back hooks

### Weeks 19–24: S4 — Butterfly Sweep to Mount

**Fieldboard**: OGRD.Butterfly.CTRL → MNT.SWP → MNT.CTRL

**Prerequisites**: S1

**Week breakdown**:
  - Week 19: CONTROL — Double underhooks seated, head on chest side
  - Week 20: TRANSITION — Off-balance to one side → hook lift → roll over → land in mount
  - Week 21: STABILIZATION — Land in mount, immediately establish base
  - Weeks 22–24: DRILLING + positional sparring (this road only)

**Duality cycle**: Attack = butterfly sweep. Defense = butterfly passing, mount escape.

**Failure**: safe — Still seated in butterfly (safe — re-grip, re-try)
**Mechanics learned**: underhook, butterfly hook, off-balance, mount base

### Weeks 25–30: S6 — Side Control to Mount Progression

**Fieldboard**: SCTR.CTRL → SCTR.KOB.TRZ → MNT.TRZ → MNT.CTRL

**Prerequisites**: S1

**Week breakdown**:
  - Week 25: CONTROL — Crossface + chest pressure in side control; Knee pressure in KOB
  - Week 26: TRANSITION — Side → KOB: slide knee onto belly. KOB → Mount: swing far leg over
  - Week 27: STABILIZATION — Each position is stable — can pause and consolidate
  - Weeks 28–30: DRILLING + positional sparring (this road only)

**Duality cycle**: Attack = positional advance SC→KOB→MNT. Defense = side escape, KOB escape, mount escape.

**Failure**: safe — Falls back to side control (safe)
**Mechanics learned**: hip pressure, knee ride, weight distribution

### Weeks 31–36: S7 — Closed Guard Triangle Choke

**Fieldboard**: CGRD.CTRL → CGRD.TRZ → CGRD.SUB

**Prerequisites**: S1

**Week breakdown**:
  - Week 31: CONTROL — Posture break, wrist isolation
  - Week 32: TRANSITION — Push one arm across → hips up → leg over far shoulder → lock ankles
  - Week 33: STABILIZATION — Lock triangle figure-four before finishing
  - Week 34: SUBMISSION — Cut angle → pull head down → squeeze knees
  - Weeks 35–36: DRILLING + positional sparring (this road only)

**Duality cycle**: Attack = triangle from guard. Defense = posture, triangle defense, stack pass.

**Failure**: safe — Still in closed guard (safe)
**Mechanics learned**: hip elevation, angle cutting, leg dexterity

### Weeks 37–42: S8 — Mount Cross Collar Choke

**Fieldboard**: MNT.CTRL → MNT.SUB

**Prerequisites**: S1

**Week breakdown**:
  - Week 37: CONTROL — High mount — walk knees to armpits
  - Week 38: STABILIZATION — Secure first grip deep in collar before attacking
  - Week 39: SUBMISSION — First hand deep in cross collar → second hand in → squeeze elbows together
  - Weeks 40–42: DRILLING + positional sparring (this road only)

**Duality cycle**: Attack = cross collar choke from mount. Defense = mount escape, choke defense.

**Failure**: safe — Still in mount (safe)
**Mechanics learned**: mount pressure, collar grip, weight forward

### Weeks 43–48: S3 — Half Guard Pass to Side Control Americana

**Fieldboard**: HGRD.CTRL → SCTR.TRZ → SCTR.CTRL → SCTR.SUB

**Prerequisites**: S1, S6

**Week breakdown**:
  - Week 43: CONTROL — Win underhook in half guard; Crossface + chest pressure in side control
  - Week 44: TRANSITION — Knee slice: free trapped leg, slide knee across, clear to side control
  - Week 45: STABILIZATION — Flatten opponent before passing — deny underhook
  - Week 46: SUBMISSION — Pin wrist to mat → figure-four grip → paint brush away from head
  - Weeks 47–48: DRILLING + positional sparring (this road only)

**Duality cycle**: Attack = half guard pass, americana. Defense = half guard retention, side escape, americana defense.

**Failure**: safe — Still in half guard top (safe)
**Mechanics learned**: crossface, underhook, knee slice, figure-four

## Year 2: Intermediate — Branch Roads

**Goal**: Add alternative attacks from known positions, build
reaction chains, expand the decision tree at each position.

**Principle**: Moderate branching. Trap systems (if A fails → B).
Recoverable failure states. Build on spine mechanics.

### Weeks 1–6: B3 — Closed Guard Flower Sweep to Mount

**Fieldboard**: CGRD.CTRL → MNT.SWP

**Prerequisites**: S1
**Builds on**: collar grip, posture break

**Week breakdown**:
  - Week 1: CONTROL — Deep collar grip + same-side sleeve grip
  - Week 2: TRANSITION — Pull collar → kick far leg over → roll into mount
  - Weeks 3–6: Chain drilling — connect this branch to its spine road

**Duality cycle**: Attack = flower sweep. Defense = base and posture in guard.
**Failure**: safe
**New mechanics**: collar grip, leg dexterity, momentum

### Weeks 7–12: B6 — Turtle to Back Take

**Fieldboard**: TRTL.CTRL → BCTR.TRZ → BCTR.SUB

**Prerequisites**: S2
**Builds on**: seatbelt, hook insertion

**Week breakdown**:
  - Week 7: CONTROL — Chest-to-back contact, seatbelt established
  - Week 8: TRANSITION — Seatbelt grip → hip to mat → insert bottom hook → top hook
  - Week 9: SUBMISSION — Hand fight to clear collar grip → slide arm under chin → lock RNC
  - Weeks 10–12: Chain drilling — connect this branch to its spine road

**Duality cycle**: Attack = turtle back take. Defense = turtle defense (granby, sit-out, standup).
**Failure**: recoverable
**New mechanics**: seatbelt, hook insertion, chair sit

### Weeks 13–18: B7 — Open Guard Pass to Side Control

**Fieldboard**: OGRD.CTRL → SCTR.TRZ → SCTR.CTRL

**Prerequisites**: S5, S6
**Builds on**: guard retention, leg clearing

**Week breakdown**:
  - Week 13: CONTROL — Grip fighting standing — control pants/ankles; Crossface in side control
  - Week 14: TRANSITION — Grip ankles → push legs to one side → pass to opposite side → settle
  - Weeks 15–18: Chain drilling — connect this branch to its spine road

**Duality cycle**: Attack = open guard pass. Defense = guard retention, re-guarding.
**Failure**: safe
**New mechanics**: toreando grip, leg clearing, pressure passing

### Weeks 19–24: B5 — Kimura Sweep from Closed Guard

**Fieldboard**: CGRD.CTRL → CGRD.TRZ → MNT.SWP

**Prerequisites**: S1, S7
**Builds on**: posture break, hip escape, figure-four lock

**Week breakdown**:
  - Week 19: CONTROL — Posture break, isolate wrist
  - Week 20: TRANSITION — Hip out to kimura side → use grip leverage to sweep → land on top
  - Weeks 21–24: Chain drilling — connect this branch to its spine road

**Duality cycle**: Attack = kimura sweep. Defense = hand-on-mat defense, posture.
**Failure**: safe
**New mechanics**: kimura grip (reusable — appears in half guard, side control), hip escape

### Weeks 25–30: B1 — Half Guard Underhook Sweep to Mount

**Fieldboard**: HGRD.CTRL → MNT.SWP

**Prerequisites**: S3
**Builds on**: underhook, hip movement

**Week breakdown**:
  - Week 25: CONTROL — Win underhook — get to hip, head under chin
  - Week 26: TRANSITION — Come to knees → drive forward → sweep over trapped leg → land in mount
  - Weeks 27–30: Chain drilling — connect this branch to its spine road

**Duality cycle**: Attack = underhook sweep from HG. Defense = whizzer counter, HG top control.
**Failure**: recoverable
**New mechanics**: underhook, hip movement, drive

### Weeks 31–36: B2 — Half Guard to Side Control to Back Control

**Fieldboard**: HGRD.CTRL → SCTR.TRZ → BCTR.TRZ → BCTR.SUB

**Prerequisites**: S3, S2
**Builds on**: crossface, seatbelt, knee slice

**Week breakdown**:
  - Week 31: CONTROL — Crossface in half guard top; Chest pressure in side control
  - Week 32: TRANSITION — Pass half → settle side control → opponent turns → slide hooks in
  - Week 33: SUBMISSION — Seatbelt → hand fight → sink RNC
  - Weeks 34–36: Chain drilling — connect this branch to its spine road

**Duality cycle**: Attack = pass→side→back chain. Defense = HG retention, side escape, back escape.
**Failure**: safe
**New mechanics**: passing, back take timing, seatbelt

### Weeks 37–42: B4 — Side Control Arm Triangle

**Fieldboard**: SCTR.CTRL → SCTR.TRZ → SCTR.SUB

**Prerequisites**: S3, S6
**Builds on**: crossface, weight distribution

**Week breakdown**:
  - Week 37: CONTROL — Heavy crossface — opponent's chin forced to far side
  - Week 38: TRANSITION — Walk around to opposite side or hop to mount → squeeze
  - Week 39: SUBMISSION — Gable grip → chest to floor → squeeze elbows
  - Weeks 40–42: Chain drilling — connect this branch to its spine road

**Duality cycle**: Attack = arm triangle from side. Defense = frame defense, arm extraction.
**Failure**: safe
**New mechanics**: crossface, head-arm control, squeeze mechanics

### Weeks 43–48: B8 — Closed Guard to Side Control Sweep

**Fieldboard**: CGRD.CTRL → SCTR.SWP → SCTR.CTRL → SCTR.SUB

**Prerequisites**: S1, S3
**Builds on**: bridge, crossface

**Week breakdown**:
  - Week 43: CONTROL — Posture break in guard; Crossface in side control
  - Week 44: TRANSITION — Overhook + far leg → bridge and roll → land in side control
  - Week 45: SUBMISSION — Pin wrist → figure-four → paint brush
  - Weeks 46–48: Chain drilling — connect this branch to its spine road

**Duality cycle**: Attack = sweep to side control, americana. Defense = guard posture, side escape.
**Failure**: safe
**New mechanics**: bridge and roll, crossface, americana

## Year 3: Advanced — Network Roads

**Goal**: Connect roads into adaptive systems. Read opponent
reactions. Switch between attacks. Dynamic path selection.

**Principle**: High branching. Opponent modeling. Decision trees.
Intentional baiting. Every network road synthesizes multiple spine+branch roads.

### Weeks 1–7: N2 — Mount Attack Cycle: Choke-Armbar Dilemma

**Fieldboard**: MNT.CTRL → MNT.SUB

**Prerequisites**: S1, S8
**Synthesizes**: mount stabilization, arm isolation, collar grip

**Week breakdown**:
  - Weeks 1–2: Review component roads — drill each branch independently
  - Weeks 3–4: Decision tree drilling — coach calls reactions, student reads
  - Weeks 5–7: Live positional sparring with network constraints

**Duality cycle**: Attack = mount attack cycle. Defense = mount escape becomes urgent.
**Failure**: safe

### Weeks 8–14: N3 — Back Control Attack System

**Fieldboard**: BCTR.CTRL → BCTR.SUB

**Prerequisites**: S2, B6
**Synthesizes**: seatbelt, hook retention, hand fighting

**Week breakdown**:
  - Weeks 8–9: Review component roads — drill each branch independently
  - Weeks 10–11: Decision tree drilling — coach calls reactions, student reads
  - Weeks 12–14: Live positional sparring with network constraints

**Duality cycle**: Attack = back control attack system. Defense = back escape system.
**Failure**: safe

### Weeks 15–21: N6 — Open Guard to Standing Takedown to Dominant

**Fieldboard**: OGRD.CTRL → OGRD.Standing.TRZ → SCTR.TRZ / MNT.TRZ

**Prerequisites**: S5, B7
**Synthesizes**: guard retention, grip fighting, off-balance

**Week breakdown**:
  - Weeks 15–16: Review component roads — drill each branch independently
  - Weeks 17–18: Decision tree drilling — coach calls reactions, student reads

**Duality cycle**: Attack = standup to takedown. Defense = takedown defense, guard pulling.
**Failure**: recoverable

### Weeks 22–28: N1 — Closed Guard Attack Tree: Sweep-or-Submit

**Fieldboard**: CGRD.CTRL → CGRD.TRZ → MNT.SWP / CGRD.SUB

**Prerequisites**: S1, S7, B5
**Synthesizes**: posture break, hip bump, figure-four lock, hip elevation

**Week breakdown**:
  - Weeks 22–23: Review component roads — drill each branch independently
  - Weeks 24–25: Decision tree drilling — coach calls reactions, student reads
  - Weeks 26–28: Live positional sparring with network constraints

**Duality cycle**: Attack = guard attack decision tree. Defense = all guard defenses simultaneously.
**Failure**: safe

### Weeks 29–35: N4 — Half Guard Passing Network

**Fieldboard**: HGRD.CTRL → SCTR.TRZ / MNT.TRZ / BCTR.TRZ

**Prerequisites**: S3, B1
**Synthesizes**: crossface, underhook, knee slice, off-balance

**Week breakdown**:
  - Weeks 29–30: Review component roads — drill each branch independently
  - Weeks 31–32: Decision tree drilling — coach calls reactions, student reads
  - Weeks 33–35: Live positional sparring with network constraints

**Duality cycle**: Attack = half guard passing network. Defense = half guard bottom game.
**Failure**: recoverable

### Weeks 36–42: N7 — Positional Ladder: Guard → Side → Mount → Back → Finish

**Fieldboard**: HGRD.CTRL → SCTR.TRZ → MNT.TRZ → BCTR.TRZ → BCTR.SUB

**Prerequisites**: S3, S6, B2, N3
**Synthesizes**: crossface, mount stabilization, seatbelt, knee slice

  - Weeks 40–42: Live positional sparring with network constraints

**Failure**: safe

### Weeks 43–49: N5 — Guard Pull to Sweep-or-Submit Chain

**Fieldboard**: OGRD.Standing.CTRL → CGRD.TRZ → CGRD.SWP / CGRD.SUB

**Prerequisites**: S1, S5, N1
**Synthesizes**: grip fighting, guard pulling, posture break

**Week breakdown**:
